# Andreii
Репозиторий посвящён исследованию ухода за кожей и косметических продуктов: составы, активные ингредиенты, принципы подбора средств, пользовательский опыт и особенности e-commerce в beauty-сфере. Материалы направлены на систематизацию знаний и практическое применение в онлайн-проектах косметической тематики
